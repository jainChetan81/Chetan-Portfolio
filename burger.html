Burger Builder

This was a project I started back in June 2020 when I had quite mastered ReactJS. I was the sole creator/developer for the first year of the project.

The system stored few sample ingredient data for the entire project in Firebase and provides an interface for deliveries and authentication

After customizing your order the page then takes you to a different section depending you have loggedin or not where you can check the price and buy the product

Customers has the ability to view all the orders

Technologies:
  • - Javascript
  • - React JS
  • - Redux JS
  • - Firebase Deploy
  • - Firebase Realtime Database
  • - Axios
